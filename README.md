# AI-Powered Durham College Query Bot

An AI assistant that allows users to search Durham College data (admissions, calendars, tuition) using natural language queries.

## Features
- Vector search with FAISS
- Handles PDFs, JSON, DOCX, and TXT files
- Command-line search tool

## Tech Stack
- Python
- FAISS
- PyMuPDF, Pandas

## How to Run
```bash
python query_gemini.py

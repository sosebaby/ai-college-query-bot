# AI College Query Bot

A **Retrieval-Augmented Generation (RAG) system** built using **LangChain** and **OpenAI / Hugging Face models** to intelligently query unstructured documents like PDFs, DOCX, JSON, and TXT files.

---

## Features

- **Vector Search with FAISS:** Retrieves the most relevant sections from large document collections.
- **Evaluation Pipeline:** Reduces hallucinations and ensures model robustness.
- **Multi-Document Support:** Works seamlessly with multiple file types and sources.
- **Extensible Architecture:** Easily adaptable to new datasets or AI models.
- **User-Friendly Interface:** Quickly get answers from large volumes of text.
